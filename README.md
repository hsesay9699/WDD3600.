## How to run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open in your browser:
   - Shop page: http://localhost:3000/
   - Add Product: http://localhost:3000/admin/add-product
